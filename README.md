# المعيار

تقرير الكتروني يقدم لمحة عن الأداء الحكومي في سوريا بشكل مختصر للارتقاء بالواقع

## الهدف
- متابعة أداء الحكومة السورية في مختلف القطاعات
- تقرير مختصر ومبسط لسهولة الفهم والمشاركة وإثراء النقاش العام
- تقديم نظرة شاملة لأبرز السلبيات والإيجابيات في مكان واحد
- يعتمد التقرير على مشاركة المجتمع ويجري تحديثه باستمرار

## المحتوى
- الإيجابيات والسلبيات في مختلف القطاعات
- الوعود الحكومية
- تجميع إحصائيات ودراسات من مختلف المصادر

## كيفية الاستخدام

يمكن للزوار:
- زيارة الموقع والإطلاع على المحتوى: https://almiyaar.nasij.org
- تقديم اقتراحات لتعديل أو إضافة نقاط جديدة: https://almiyaar.nasij.org/contribute

## تطوير المشروع

- تثبيت [Node.js](https://nodejs.org/en/download/) (يفضل الإصدار 18 أو أحدث)
- تثبيت مستلزمات المشروع:

```sh
npm install
```

- تجربة الموقع على الجهاز:

```sh
npm run dev -- --open
```

- بناء الموقع:

```sh
npm run build
```
